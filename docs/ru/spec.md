# OpenAPI спецификация сервисов DaData

> _If you're interested in the English version of this spec, let us know [on the GitHub](https://github.com/alexchexes/dadata-sdk)._

## Описание спецификации

DaData — это сервис подсказок и поиска по российским адресам, организациям, ФИО и другим данным.

Данная спецификация описывает все публично доступные API «Дадаты», и включает полное описание формата запросов и ответов API, во многих случаях – с пояснениями и примерами использования.

_Спецификация не является официальным продуктом «ДаДаты»_.

## Аутентификация

- Запросы к API подсказок и «Поиск по коду» требуют передать API-ключ (токен) в заголовке `Authorization: Token <API_KEY>`. [Получить API-ключ](https://dadata.ru/profile/#info)
- Запросы к API «Стандартизации», а также данным об аккаунте, помимо токена, требуют также указать секретный ключ в заголовке `X-Secret: <SECRET_KEY>`
  **Внимание!** Запросы к «Стандартизации» и данным аккаунта намеренно недоступны из браузера.

Некоторые сервисы являются платными. Для уточнения цен и условий обратитесь к [соответствующему разделу](https://dadata.ru/pricing/) официального сайта.

## Лимиты запросов

### API «Подсказки» (`suggest`) и «Поиск по коду» (`findById`):

- До **30 запросов в секунду** на один IP
- До **60 новых соединений в минуту** на один IP
- Суточные лимиты зависят от тарифного плана.

### API стандартизации:

- До **20 запросов в секунду** на один IP
- До **60 новых соединений в минуту** на один IP
