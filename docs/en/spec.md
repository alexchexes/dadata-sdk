title: OpenAPI spec for Dadata API
---

::: info
This documentation was developed in 2025 when the official OpenAPI spec had not yet been published.
Since then (as of March 2026), much has changed, and although this unofficial documentation is still useful, it requires updating. If you wish to help, join us [on GitHub](https://github.com/alexchexes/dadata-sdk).

---

Official OpenAPI Schemas:</br>

1. [cleaner.yml](https://dadata.ru/files/openapi/cleaner.yml) — "Standardization". Methods clean/address, clean/phone, and similar. </br>
2. [suggestions.yml](https://dadata.ru/files/openapi/suggestions.yml) — "Suggestions" and search. Methods suggest, findById, geolocate, and others. </br>
3. [profile.yml](https://dadata.ru/files/openapi/profile.yml) — personal account. Balance, statistics, versions.

:::

::: warning

The documentation is in the early stage of development. The site might change frequently, but we will try to make it as user-friendly as possible.

[Click here](https://github.com/alexchexes/dadata-sdk) to go to GitHub if you want to help.

:::
